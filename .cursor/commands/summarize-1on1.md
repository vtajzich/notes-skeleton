# Summarize 1-on-1

Update the Summary.md for a 1-on-1 meeting series with any unsummarized meetings.

## Instructions

1. Read the existing `Summary.md` in the target 1-on-1 folder
2. Check the `## Summarized Meetings` ledger for already-processed dates
3. Find any meeting notes in the folder that are NOT in the ledger
4. Read each unsummarized meeting note
5. Update Summary.md by merging new information:
   - Update Hot Topics (add new, mark resolved)
   - Merge Action Items (add new open, mark completed)
   - Update Topic History table
   - Add new dates to Summarized Meetings ledger

## Rules
- Summary.md is the canonical location for open tasks in this series
- Do not duplicate — merge and update
- Preserve existing content that is still relevant
- Use action-items rule for task format
