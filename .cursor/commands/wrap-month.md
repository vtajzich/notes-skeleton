# Wrap Month

Generate a monthly wrap from the weekly wraps of the target month.

## Instructions

1. Find all weekly wraps in `wraps/YYYY/MM/Www/week.md` for the target month
2. Read each weekly wrap
3. Generate a monthly wrap at `wraps/YYYY/MM/month.md`

## Output Structure

```markdown
# Month YYYY
#work/wraps

## Month Summary
Narrative paragraph covering key themes and outcomes.

## Highlights
Bullet list of the most significant events and achievements.

## Open Action Items
Table of all open items carried forward.

## Major Themes
H3 subsections with narrative for each theme (2-4 themes).

## Decisions
Table with Date, Decision, Context.

## Weekly Wrap Sources
Links to each weekly wrap consumed.
```
