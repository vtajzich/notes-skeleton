# Wrap Quarter

Generate a quarterly wrap from the monthly wraps.

## Instructions

1. Find all monthly wraps in `wraps/YYYY/MM/month.md` for the target quarter
2. Read each monthly wrap
3. Generate a quarterly wrap at `wraps/YYYY/Qx.md`

## Output Structure

```markdown
# Qx YYYY
#work/wraps

## Executive Summary
3-5 sentence strategic narrative.

## Quarter Highlights
Bullet list of most significant outcomes.

## Key Metrics
Table showing start vs end of quarter metrics.

## Strategic Themes
H3 subsections with multi-paragraph narrative per theme.

## Decisions
Table with Date, Decision, Impact.

## Looking Ahead
Priorities and expectations for next quarter.

## Monthly Wrap Sources
Links to each monthly wrap consumed.
```
